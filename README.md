# 🔐 Gerador de Senhas Seguras

Um gerador de senhas interativo e moderno com avaliação de força em tempo real.

## 🎯 Funcionalidades

- **Geração Personalizada**: Controle o tamanho da senha (6-32 caracteres)
- **Opções Flexíveis**: Escolha quais tipos de caracteres incluir:
  - Letras maiúsculas (A-Z)
  - Letras minúsculas (a-z)
  - Números (0-9)
  - Caracteres especiais (!@#$%^&*)
- **Avaliação de Força**: Indicador visual e textual da força da senha
- **Copiar com um Clique**: Botão para copiar a senha para a área de transferência
- **Design Responsivo**: Funciona em dispositivos móveis e desktop
- **Interface Moderna**: Design limpo e intuitivo com animações suaves

## 🚀 Como Usar

1. **Ajuste o Tamanho**: Use o slider para definir o comprimento da senha
2. **Selecione os Caracteres**: Marque/desmarque as opções de caracteres
3. **Gere a Senha**: Clique em "Gerar Nova Senha" ou aguarde a atualização automática
4. **Copie a Senha**: Clique no botão 📋 para copiar a senha gerada

## 💻 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Estilização moderna com animações e responsividade
- **JavaScript (ES6+)**: Lógica de geração e avaliação de senhas

## 📋 Pré-requisitos

Nenhum! Basta abrir o arquivo `index.html` em qualquer navegador moderno.
